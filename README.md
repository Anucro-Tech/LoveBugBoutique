# Lovebug Boutique Co. starter site

A simple static website for Lovebug Boutique Co. designed for GitHub Pages.

## Files

- `index.html` — page content
- `styles.css` — layout and design
- `assets/lovebug-logo.png` — logo artwork
- `assets/favicon.svg` — simple ladybug browser icon
- `CNAME` — custom domain for GitHub Pages

## Edit these first

1. Replace placeholder product illustrations with real boutique photos.
2. Confirm pickup, delivery, shipping, and service area wording.
3. Add phone number or Facebook link if Rachel wants them listed.
4. Replace placeholder categories with actual inventory categories.
5. Add prices only if Rachel wants shoppers to see pricing online.
