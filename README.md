# Lovebug Boutique Co. starter site

A simple static website for Lovebug Boutique Co. designed for GitHub Pages.

## Files

- `index.html` — page content
- `styles.css` — layout and design
- `assets/lovebug-logo.png` — cropped logo from provided artwork
- `assets/favicon.svg` — simple ladybug browser icon
- `CNAME` — custom domain for GitHub Pages

## Edit these first

1. Replace placeholder arrangement illustrations with real arrangement photos.
2. Confirm pickup/delivery wording.
3. Add service towns if Rachel wants them listed.
4. Confirm whether the Instagram link is live.
5. Add pricing only after Rachel approves the ranges.

## GitHub Pages quick deploy

1. Create a new GitHub repository, for example `lovebug-boutique-site`.
2. Upload these files to the repo root.
3. Go to Settings > Pages.
4. Source: Deploy from a branch.
5. Branch: `main` and folder `/root`.
6. Save.
7. After the domain is purchased and DNS is set, GitHub Pages should use `lovebugboutiqueco.com` from the `CNAME` file.
